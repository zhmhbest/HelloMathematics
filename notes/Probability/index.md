## 排列组合
### 排列数
$$\mathop{A}_{n}^{k} = \dfrac{n!}{(n-k)!}$$
### 组合数
$$\mathop{C}_{n}^{k} = \dfrac{\mathop{A}_{n}^{k}}{k!} = \dfrac{n!}{k!(n-k)!}$$



## 事件关系及运算
### 运算规则
- $P(A∪B) = P(A) + P(B) - P(AB)$
- $1-P(A∪B) = P(\overline{A∪B}) = P(\overline{A}∩\overline{B})$
- $1-P(A∩B) = P(\overline{A∩B}) = P(\overline{A}∪\overline{B})$
- 减等并对：$P(A-B)=P(A\overline{B})=P(A)-P(AB)$
- 对立事件：$P(\overline{A})=1-P(A)$
- 独立事件：$P(AB)=P(A) \cdot P(B)$
### 例题
#### $eg1$
在一个箱子中共有7个球，3个黑球，四个白球：
（1）**无放回**抽取3个球，取得两黑一白（事件$A$）的概率。
$$P(A)= \dfrac
    {\mathop{C}_{3}^{2} \mathop{C}_{4}^{1}}
    {\mathop{C}_{7}^{3}}
    = \dfrac{12}{35}
$$
（2）**有放回**抽取3个球，取得两黑一白（事件$B$）的概率。
$$P(B)=
    \mathop{C}_{3}^{2}
    ×
    \left( \dfrac{3}{7} \right)^2
    ×
    \dfrac{4}{7}
$$



## 全概率及贝叶斯公式

### 运算规则
- 条件概率：$P(A|B)=\dfrac{P(AB)}{P(B)}$
- 联合概率：$P(AB)$或$P(A∩B)$
- 乘法公式：$P(AB)=P(A|B) \cdot P(B)=P(B|A) \cdot P(A)$
- 全概率公式：$P(A) = \sum\limits_{i=1}^{n}P(B_i)P(A|B_i)$
- 贝叶斯公式：$P(B_iA) = P(AB_i) ⇒ P(B_i|A) = \dfrac{P(AB_i)}{P(A)} = \dfrac{P(A|B_i)P(B_i)}{P(A)}$

### 条件概率与联合概率
条件概率与联合概率的主要区别在于两者对于事件1的

### 例题
#### $eg1$
某家庭有3个小孩，且至少有一个是女孩，求该家庭至少有一个男孩的概率。
即求在至少有一个是女孩（事件$A$）的情况下，至少有一个是男孩（事件$B$）的概率。
$P(A) = 1 - P(\overline{A}) = 1 - \dfrac{1}{8} = \dfrac{7}{8}$（不是全男）
$P(BA) = 1 - 2 × \dfrac{1}{8} = \dfrac{6}{8}$（不是全男或全女）
$P(B|A) = \dfrac{P(BA)}{P(A)} = \dfrac{6}{7}$
#### $eg2$
盒中有4个红球，6个黑球，随机取出一球，观察后放回，并追加2个同色，再从盒中第二次抽取一球，求：
（1）第二次取出的是黑球（事件$A$）的概率。

设第一次取出红球为事件$B_1$，第一次取出是黑球为事件$B_2$：
$P(B_1) = \dfrac{4}{10}$、$P(B_2) = \dfrac{6}{10}$
$P(A|B_1) = \dfrac{6}{12}$、$P(A|B_2) = \dfrac{8}{12}$
$P(A) = P(B_1)⋅P(A|B_1) + P(B_2)⋅P(A|B_2) = \dfrac{3}{5}$

（2）第二次取出是黑球时，第一次取出也是黑球的概率。

$P(B_2|A) = \dfrac{P(B_2)⋅P(A|B_2)}{P(A)} = \dfrac{2}{3}$



## 随机变量

#### $eg1$
盒中有6个球，4白2黑，任取其2：
设抽到的白球数为$X$
$P{X=0} = \dfrac{C_4^0⋅C_2^2}{C_6^2} = \dfrac{1}{15}$
$P{X=1} = \dfrac{C_4^1⋅C_1^2}{C_6^2} = \dfrac{8}{15}$
$P{X=2} = \dfrac{C_4^2⋅C_0^2}{C_6^2} = \dfrac{6}{15}$
**分布律（表格）**
| $X$ | 0 | 1 | 2 |
|  -  | - | - | - |
| $P$ | $\frac{1}{15}$ | $\frac{8}{15}$ | $\frac{6}{15}$ |
**分布函数**
$$
    F(x)=
    \begin{cases}
                   0  \ \ \ \ & x<0
    \\   \frac{1}{15} \ \ \ \ & 0≤x<1
    \\   \frac{9}{15} \ \ \ \ & 1≤x<2
    \\              1 \ \ \ \ & 2≤x
    \end{cases}
$$

### 概念
#### 离散型随机变量
如果随机变量$X$的取值是有限个或可列无穷个，则称$X$为离散型随机变量。
#### 离散型随机变量的分布律
$P\{X=x_n\} = p_n, \ \ \ \ (n = 1, 2, \dots)$
| $X$ | $x_1$ | $x_2$  | $\dots$  | $x_n$  |
|  -  | - | - | - | - |
| $P$ | $p_1$ | $p_2$ | $\dots$ | $p_n$ |
$\sum\limits_{i=1}^{n} p_i= 1$
#### 概率密度（概率密度函数）
概率密度函数对区间积分的面积就是事件在这个区间发生的概率。
$\int\nolimits_{-∞}^{+∞}{f(t)}\,{\rm d}t = 1$



## 常用分布
### 离散型
#### 二项分布（$Binomial$ $Distribution$）
$X∼B(n, p): P\{X=k\} = C_n^k p^k(1-p)^{n-k}, \ \ \ \ (k=0,1,\dots)$
#### 伯努力分布（$Bernoulli$ $Distribution$）
$P\{X=k\} = p^k(1-p)^{1-k}, \ \ \ \ (k=0,1,\dots)$
| $X$ | $0$ | $1$  |
|  -  | - | - |
| $P$ | $1-p$ | $p$ |
#### 泊松分布 （$Poisson$ $Distribution$）
$X∼P(λ):P\{X=k\} = \dfrac{λ^k}{k!}e^{-λ}, \ \ \ \ (λ>0; k=0,1,2,\dots)$
#### 几何分布（$Geometric$ $Distribution$）
$X∼GE(p):P\{X=k\} = q^{k-1}p \ \ \ \ (p≥0,q≥0,p+q=1;k=1,2,\dots)$
#### 超几何分布（$Hypergeometric$ $Distribution$）
$P\{X=k\} = \dfrac
    {\mathop{C}_{M}^{k}\mathop{C}_{N-M}^{n-k}}
    {\mathop{C}_{N}^{n}}
    \ \ \ \ (k=0,1,2,\dots,\min(M,m))$

### 连续型
#### 均匀分布（$Uniform$ $Distribution$）
$X∼U(a, b)$
**密度函数**：$f(x)=
    \begin{cases}
        \dfrac{1}{b-a} \ \ \ \ & a ≤ x ≤ b
    \\  0              \ \ \ \ & others
    \end{cases}
$
**分布函数**：$F(x)=
    \begin{cases}
        0                \ \ \ \ & x < a
    \\  \dfrac{x-a}{b-a} \ \ \ \ & a ≤ x ≤ b
    \\  1                \ \ \ \ & b < x
    \end{cases}
$
#### 指数分布（$Exponential$ $Distribution$）
$X∼E(λ)$
**密度函数**：$f(x)=
    \begin{cases}
        λe^{-λx} \ \ \ \ & x>0
    \\  0        \ \ \ \ & x≤0
    \end{cases}
$
**分布函数**：$F(x)=
    \begin{cases}
        1-e^{-λx} \ \ \ \ & x>0
    \\  0         \ \ \ \ & x≤0
    \end{cases}
$
#### 正态分布（$Normal$ $Distribution$）
$X∼N(μ, σ^2)$
**密度函数**：$f(x)=
    \dfrac{1}{\sqrt{2π}σ}e^{-\dfrac{(x-μ)^2}{2σ^2}}
    \ \ \ \ -∞ < x < +∞
$
**标准正态分布的密度函数**：$φ(x)=
    \dfrac{1}{\sqrt{2π}}e^{-\dfrac{1}{2}x^2}
$
#### 伽马分布（$Gamma$ $Distribution$）
$X∼Γ(r, λ)$

**$Γ$函数**：$Γ(r) =
    \int\nolimits_{0}^{+∞}{x^{r-1}e^{-x}}\,{\rm d}x,
    \ \ \ \ x∈(0, +∞)
$

**密度函数**：$f(x)=
    \begin{cases}
        \dfrac{λ^r}{Γ(r)}x^{r-1}e^{-λx}
            \ \ \ \ & x>0
    \\  0
            \ \ \ \ & x≤0
    \end{cases}
$

当$r=1, Γ(1)=1$得到**指数分布**；
当$r=n, Γ(n)=(n-1)!$得到**埃尔朗分布**；
当$r=\frac{n}{2}, λ=\frac{1}{2}$得到**卡方分布**；

#### 埃尔朗分布（$Erlang$ $Distribution$）
**密度函数**：$f(x)=
    \begin{cases}
        \dfrac{λ^n}{(n-1)!}x^{n-1}e^{-λx}
            \ \ \ \ & x>0
    \\  0
            \ \ \ \ & x≤0
    \end{cases}
$
#### 卡方分布（$chi-square$ $Distribution$）
**密度函数**：$f(x)=
    \begin{cases}
        \dfrac{1}{2^{\frac{n}{2}}Γ(\frac{n}{2})}
        x^{\frac{n}{2}-1}e^{-\frac{x}{2}}
            \ \ \ \ & x>0
    \\  0
            \ \ \ \ & x≤0
    \end{cases}
$



## 统计
### 期望
也就是均值，是概率加权下的“平均值”，是每次可能结果的概率乘以其结果的总和，反映的是随机变量平均取值大小。
- 连续型: $E(x) = \int\nolimits_{-∞}^{+∞}{xf(x)}\,{\rm d}x$
- 离散型: $E(x) = \sum\limits_{i}x_i p_i$
#### 期望的性质
- $E(C) = C$
- $E(Cx) = C \cdot E(x)$
- $E(x + y) = E(x) + E(y)$
- $x$、$y$相互独立$\Leftrightarrow E(xy) = E(x)E(y)$
### 方差
是衡量随机变量或一组数据时离散程度的度量，是用来度量随机变量和其数学期望之间的偏离程度。
$Var(x) = D(x) = σ^2 = \dfrac{\sum\limits_{i=1}^{n}(x_i-μ)^2}{n}$
#### 方差的性质
- $D(x) = \sum\limits_{i=1}^{n}p_i⋅(x_i-μ)^2$
- $D(x) = \int\nolimits_{a}^{b}{(x-μ)^2f(x)}\,{\rm d}x$
- $D(x) = E\left({(x-E(x))^2)}\right)=E(x^2)-(E(x))^2$
- $D(C) = 0$
- $D(Cx) = C^2D(x)$
- $D(C+x) = D(x)$
- $D(x±y) = D(x) + D(y) ± 2Cov(x, y)$
- $x$、$y$相互独立$\Rightarrow D(x±y) = D(x) ± D(y)$
#### 常见分布的期望与方差
| 分布 | 参数 | 期望 | 方差 |
| :-: | :-: | :-: | :-: |
| 两点分布 | $0<p<1$ | $p$ | $p(1-p)$ |
| 二项分布 | $n≥1 \newline 0<p<1$ | $np$ | $np(1-p)$ |
| 泊松分布 | $λ>0$ | $λ$ | $λ$ |
| 均匀分布 | $a<b$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| 指数分布 | $θ>0$ | $θ$ | $θ^2$ |
| 正态分布 | $μ,σ>0$ | $μ$ | $σ^2$ |
### 标准差
$σ = \sqrt{D(x)} = \sqrt{\dfrac{\sum(x-μ)^2}{n}}$
### 协方差
用于衡量两个变量的总体误差；当两个变量相同的情况下，协方差其实就是方差。
$Cov(x, y)
    \newline \ \ \ =E\left\{ (x - E(x)) \cdot (y - E(y))   \right\}
    \newline \ \ \ =E\left\{ xy - xE(y) - yE(x) + E(x)E(y) \right\}
    \newline \ \ \ =E(xy) - E(x)E(y)
$
#### 协方差性质
- $Cov(x, y) = Cov(y, x)$
- $Cov(ax, by) = ab ⋅ Cov(x, y)$
- $Cov(x_1 + x_2, y) = Cov(x_1, y) + Cov(x_2, y)$
- $Cov(x, y) ? 0$
  - $Cov(x, y)>0$：$x$和$y$的变化趋势**相同**
  - $Cov(x, y)=0$：$x$和$y$**不相关**
  - $Cov(x, y)<0$：$x$和$y$的变化趋势**相反**
#### 协方差矩阵
对于$n$个随机向量$(x_1,x_2,x_3,\dots,x_n)$，任意两个元素$x_i$和$x_j$都可以得到一个协方差，从而形成一个$n×n$的矩阵，该矩阵就叫做**协方差矩阵**。
$c_{ij} = Cov(x_i, x_j)$
$
    \left[\begin{array}{c}
        c_{11} & c_{12} & \dots  & c_{1n}
    \\  c_{21} & c_{21} & \dots  & c_{2n}
    \\  \vdots & \vdots & \ddots & \vdots
    \\  c_{n1} & c_{n2} & \dots  & c_{nn}
    \end{array}\right]_{m×n}
$
#### $Pearson$相关系数
协方差可以描述$x$和$y$的相关程度，但是协方差的值和$x$、$y$的值采用的是不同的量纲，导致协方差在数值上表现出比较大的差异，因此可以引入相关系数来表示$x$和$y$的相关性。
$ρ(x, y) = \dfrac{Cov(x, y)}{\sqrt{D(x)} \cdot \sqrt{D(y)}}$
$Pearson$相关系数取值范围为$[-1,1]$
| 绝对值 | 含义 |
| :-:   | :-: |
| $0.8-1.0$ | 极强相关 |
| $0.6-0.8$ | 强相关 |
| $0.4-0.6$ | 中等相关 |
| $0.2-0.4$ | 弱相关 |
| $0.0-0.2$ | 极弱相关 |

### 原点矩、中心矩
假设$x$和$y$是随机变量：
- 若$E(x^k), \ \ k=1,2,\dots$存在，则称它为$x$的$k$阶**原点矩**，简称$k$阶矩。
- 若$E\{[x-E(x)]^k)\}, \ \ k=1,2,\dots$存在，则称它为$x$的$k$阶**中心矩**。
- 若$E\{(x-c)^k)\}, \ \ k=1,2,\dots$存在，则称它为$x$关于点$c$的$k$阶矩（**原点矩**）。
- 若$E\{x^k ⋅ y^p\}, \ \ p,k=1,2,\dots$存在，则称它为$x$和$y$的$k+p$阶**原点矩**。
- 若$E\{[x-E(x)]^k ⋅ [y-E(y)]^p\}, \ \ p,k=1,2,\dots$存在，则称它为$x$和$y$的$k+p$阶**中心矩**。
#### 概念降维
- $x$的数学期望$E(x)$是$x$的一阶原点矩。
- $x$的方差$D(x)$是$x$的二阶中心矩。
- $x$和$y$的协方差$Cov(x, y)$是$x$和$y$的二阶混合中心矩。

### 峰度
又称峰态系数。表示了概率密度分布曲线在平均值处峰值高低的特征数，直观来讲，峰度反映的是峰部的尖度。
样本的峰度是和正态分布相比较而言的统计量，如果峰度值大于三，那么峰的形状比较尖，比正态分布峰要陡峭。反之亦然。
#### 计算公式
随机变量的四阶中心矩与方差平方的比值。
$kurtosis = \dfrac{\sum\limits_{i=1}^{n}(x_i-\overline{x})^4}{(n-1) \bullet s^4}$

### 偏度
是描述分布偏离对称性程度的一个特征数，当分布左右对称的时候，偏度系数为0，当偏度系数大于0时候，即重尾在右侧时，该分布为右偏；当偏度系数小于0时候，即重尾在左侧时，该分布为左偏。
#### 计算公式
随机变量的三阶中心矩与样本的平均离均差立方和的比值。
$kurtosis = \dfrac{\sum\limits_{i=1}^{n}(x_i-\overline{x})^3}{(n-1) \bullet s^3}$


## 定理
### 切比雪夫定理
$D(x)$越小，时间$\{|x-μ|<ε\}$发生的概率就越大，即：$x$取的值基本上集中在期望$μ$附近。
#### 公式
设随机变量$x$的期望为$μ$，方差为$σ^2$，对于任意的正数$ε$：
$$
    P\{|x-μ|<ε\} ≤ \dfrac{σ^2}{ε^3}
$$

### 大数定律
随着样本容量$n$的增加，样本平均数将接近于总体平均数（期望$μ$），所以在统计推断中，一般都会使用样本平均数估计总体平均数的值。

### 中心极限定理
设从均值为$μ$、方差为$σ2$有限的任意一个总体中抽取样本量为$n$的样本，当$n$充分大时，样本均值的抽样分布近似服从均值为$\frac{μ}{n}$、方差为$\frac{σ^2}{n}$的正态分布。
