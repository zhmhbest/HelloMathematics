## 定义

对于实数$R$上的线性空间$V$，有$α,β,γ∈V,k∈R$，在$V×V$上定义了实函数$(α,β)$，满足以下条件

- **对称性**：$(α,β) = (β,α)$
- **线性性**：
  - $(α+γ,β) = (α,β) + (γ,β)$
  - $(kα,β) = k(α,β)$
- **正定性**：
  - $(α,α)≥0$
  - $(α,α)=0 { \ \ \ ⇔ \ \ \ } α=0$

则称实数$(α,β)$为向量$α$与$β$的**内积**，称定义了内积的线性空间$V$为**内积空间**。
