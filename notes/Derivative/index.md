## 导数微分

$y'(x) = 
    \dfrac{
        {\rm d}y
    }{
        {\rm d}x
    }
$

$y''(x) = 
    \dfrac{
        {\rm d} (\dfrac{{\rm d}y}{{\rm d}x})
    }{
        {\rm d}x
    } = 
    \dfrac{{\rm d}^2y}{{\rm d}x^2}
$

## *Taylor*展开

$$f(x) = \dfrac{f(x_0)}{0!} + \dfrac{f'(x_0)}{1!}(x-x_0) + \dfrac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \dfrac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$
